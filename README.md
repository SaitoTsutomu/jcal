# jcal

`jcal` is a package for Japanese holiday at 2000-2027.
https://www8.cao.go.jp/chosei/shukujitsu/gaiyou.html

## Installation

```sh
uv add jcal
```

## Example

```python
from pprint import pprint
from jcal import holidays

pprint(holidays(2026))
```
